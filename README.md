
# 📋 Zen Board

**Zen Board** — это минималистичный и интуитивно понятный таск-менеджер в формате Kanban-доски. Проект создан для тех, кто ценит чистоту интерфейса и хочет сосредоточиться на задачах, не отвлекаясь на лишний функционал.

🚀 **[Посмотреть Live Demo](https://anastasiia-startseva.github.io/zen-board/)**

---

## ✨ Особенности (Features)

*   **🔄 Полный CRUD:** Удобное создание, редактирование и удаление как отдельных задач, так и целых колонок.
*   **🖱 Drag-and-Drop:** Плавное перемещение карточек между этапами выполнения (реализовано на `@dnd-kit`).
*   **💾 Persistence:** Автоматическое сохранение всех изменений в `localStorage` — ваши данные не пропадут после перезагрузки страницы.
*   **📱 Responsive Design:** Интерфейс полностью адаптирован под мобильные устройства, планшеты и десктопы.
*   **🧘 Clean UI:** Лаконичный "Zen" дизайн с фокусом на UX и отсутствие визуального шума.

## 🧠 Технические решения и вызовы

> Этот раздел описывает мой подход к разработке и архитектурные решения.

*   **Оптимизация Drag-and-Drop:** Я выбрала библиотеку `@dnd-kit` за её модульность и легковесность по сравнению с аналогами. В проекте реализована сложная логика сортировки внутри колонок и перемещения между ними с использованием стратегии `verticalListSortingStrategy`.
*   **Управление состоянием:** Для данного масштаба приложения я намеренно отказалась от тяжелых стейт-менеджеров (вроде Redux/Zustand) в пользу нативного **React State**. Архитектура компонентов спроектирована так, чтобы минимизировать *Prop Drilling*.
*   **Типизация данных:** Весь проект написан на **TypeScript**. Описаны интерфейсы для сущностей `Task`, `Column` и типов `Id`, что гарантирует отсутствие ошибок при манипуляции данными и упрощает поддержку кода.
